## 📖 项目简介

HearMeGLM是我本学期大语言模型课程的期末项目，目标是通过微调ChatGLM-6B基础大语言模型，实现一个能够扮演心理咨询师角色的虚拟咨询师。这个项目不仅仅是为了完成作业，更是我对心理健康问题进行深思的尝试，也体现了作为一名AI专业学生，我希望为心理健康事业贡献自己的一份力量。我希望HearMeGLM能够启发更多关于AI如何支持心理健康的思考。
